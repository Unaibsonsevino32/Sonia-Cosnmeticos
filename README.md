# 💄 Sônia Cosméticos

Este projeto consiste em um site institucional desenvolvido para a **Sônia Cosméticos**, com o objetivo de apresentar seus produtos de beleza de forma moderna, organizada e atrativa. O site oferece uma experiência de navegação intuitiva, permitindo que os clientes conheçam a variedade de cosméticos disponíveis e entrem em contato de maneira rápida e prática.

O projeto foi criado com foco em um design elegante, responsividade e boa experiência do usuário, garantindo uma navegação eficiente em computadores, tablets e smartphones.

## 🚀 Funcionalidades

* Página inicial com destaque para os principais produtos;
* Catálogo de cosméticos organizado por categorias;
* Seção institucional sobre a empresa;
* Área com produtos em destaque e promoções;
* Informações de contato;
* Botão de atendimento via WhatsApp;
* Layout totalmente responsivo.

## 🛠️ Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Flexbox
* CSS Grid

## 🎯 Objetivo do Projeto

Este projeto foi desenvolvido para colocar em prática conhecimentos em Desenvolvimento Front-End, aplicando boas práticas de HTML semântico, estilização com CSS, responsividade e JavaScript para criar uma interface moderna, funcional e visualmente agradável.

## 📱 Responsividade

O site foi desenvolvido utilizando técnicas de design responsivo, proporcionando uma experiência de navegação consistente em diferentes tamanhos de tela.
